What is port and usage :

  • Port is software defined number which is associated with specific protocol that is used for communication between 2 devices in the network for specific service/process.
  • These ports are not visible and can not touch manually.

Purpose: Ports help manage multiple network services and applications on a single device by distinguishing between different types of network traffic.

Types of ports:

  • Ports are identified by port numbers, which are 16-bit integers ranging from 0 to 65,535 into 3 categories.

    1. Well-Known Ports: Ranges from 0 to 1,023. These ports are assigned and controlled by the ** ** (IANA) and are used by widely used protocols and services. For example:
    • Port 80: HTTP (web traffic)
    • Port 443: HTTPS (secure web traffic)
    • Port 25: SMTP (email sending)
    • Port 22: SSH (secure shell)

    2.Registered Ports: Ranges from 1,024 to 49,151. These ports are used by software applications that are not as universally recognized as those using well-known ports. They can be registered with IANA for specific applications or services.

  1. Dynamic or Private Ports: Ranges from 49,152 to 65,535. These ports are typically used for ephemeral or temporary purposes and are often assigned dynamically by the operating system for client-side applications to use during communication sessions.
